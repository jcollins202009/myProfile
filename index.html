Robofriends

This React application gets users using jsonplaceholder API, and then displays their names in cards on the website. Each card gets a robot image using robohash API. The application also allows you to narrow down the list of robot cards displayed by searching for part of their name in a search box.

SmartBrain

This is a React application that allows users to enter an image url of a person, and the application will perform facial recognition on the image. It also has signin and signoff capability. As well the application keeps track of who has performed the most facial recognitions.
